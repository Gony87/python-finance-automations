# AI CFO — n8n Workflow

An AI-powered CFO assistant built in n8n using OpenAI GPT-4.

## Tools
- **FP&A Tool** — Financial planning and analysis
- **Treasury Tool** — Cash and liquidity management  
- **T&E Tool** — Travel and expense management
- **Accounting Tool** — General accounting queries

## How to use
1. Import `AI_CFO.json` into your n8n instance
2. Configure your OpenAI API key
3. Start chatting with your AI CFOd
